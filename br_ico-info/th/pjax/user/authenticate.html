การระบุตัวตน

บัญชีของท่านได้รับการยืนยันตัวตนอย่างสมบูรณ์แล้ว ท่านสามารถเรียกดู วงเงินซื้อขายของท่านที่นี่

ต้องการตรวจสอบอายุของบัญชีนี้ โปรดติดต่อ ฝ่ายบริการลูกค้า สำหรับข้อมูลเพิ่มเติม

รับรองความถูกต้องของบัญชีของคุณ โดยการตรวจสอบตัวตนและที่อยู่ของคุณ

1. หลักฐานแสดงตัวบุคคล

We accept:
  • Passport
  • ใบขับขี่
  • National ID card or any government issued document which contains a photo, your name, and date of birth
Requirements:
  • Must be a clear, colour photo or scanned image
  • Still valid
  • Only JPG, JPEG, GIF, PNG and PDF formats are accepted
  • Maximum upload size for each file is 3MB
